# Get Started with Flood

Welcome to Flood, a powerful toolkit designed to streamline your Flutter and Dart development process. With its modular approach, Flood simplifies every aspect of development, from styling to authentication, making it suitable for projects of any scale. Let's explore how you can start using Flood in your projects.

## Explore Flood's Potential

Before diving into using Flood, you may want to explore its capabilities and see if it aligns with your project requirements. Flood offers a range of resources to help you understand its features and potential:

1. **Home Page**: Check out the [Home page for Flood](https://www.flooddev.com) to watch videos of the key modules of Flood, showcasing its top features.

2. **Guides**: Follow our comprehensive guides, starting with [Install & Run Your First App](/install-and-run-your-first-app), to learn how to build and release a Flood app step-by-step. These guides will walk you through the process of creating a complete application using Flood.

3. **Example Todo App**: Flood provides an example Todo App that showcases its implementation in a practical scenario. You can find the source code of the example app here: [Flood Example App](https://github.com/JLogical-Apps/flood-example). Explore the code to see how Flood is used in a real-world application.

4. **Video Walkthrough**: Watch [this YouTube video](https://www.youtube.com/watch?v=I-s2sBPGPwQ) of Jake Boychenko, the creator of Flood, build and explain an entire Todo app from scratch using Flood.

5. **Flood in Production**: To see Flood's effectiveness in production-ready applications, you can check out the list of apps built and released using Flood. Visit [JLogical Projects](https://www.jlogical.com/projects) to see the success stories of Flood in action.

6. **Flood's GitHub Repository**: Take a look at [Flood's GitHub repository](https://github.com/JLogical-Apps/flood) to get a deeper understanding of its implementation.

By exploring these resources, you can gain a better understanding of Flood's capabilities and determine if it's the right fit for your project.
