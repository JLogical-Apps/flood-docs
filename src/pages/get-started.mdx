# Get Started with Flood

Welcome to Flood, a powerful framework designed to streamline your Flutter and Dart development process. With its modular approach, Flood simplifies every aspect of development, from styling to authentication, making it suitable for projects of any scale. Let's explore how you can start using Flood in your projects.

## Explore Flood's Potential

Before diving into using Flood, you may want to explore its capabilities and see if it aligns with your project requirements. Flood offers a range of resources to help you understand its features and potential:

1. **Open Documentation**: The [documentation for Flood](/) is openly available, allowing you to explore the framework's concepts, modules, and APIs in detail. Take a look at the documentation to get a comprehensive understanding of what Flood has to offer.

2. **Example Todo App**: Flood provides an example Todo app that showcases its implementation in a practical scenario. You can find the source code of the example app here: [Flood Example App](https://github.com/JLogical-Apps/flood-example). Explore the code to see how Flood is used in a real-world application.

3. **Flood in Production**: To see Flood's effectiveness in production-ready applications, you can check out the list of apps built and released using Flood. Visit [JLogical Projects](https://www.jlogical.com/projects) to see the success stories of Flood in action.

By exploring these resources, you can gain a better understanding of Flood's capabilities and determine if it's the right fit for your project.

Currently, Flood is available through a **closed beta** testing program, accessible via sponsorship of [JLogical](https://jlogical.com/) on GitHub. Sponsoring JLogical not only grants you access to Flood's extensive features but also provides you with additional benefits:

To sponsor JLogical and gain access to Flood, follow these steps:

## Use the Flood Template

After gaining access to Flood, you can quickly get started with your project using the Flood template. The template is designed to simplify project setup and ensure you're on the right track from the beginning.

To use the Flood template, follow these steps:

1. Clone the Flood template from GitHub: [Flood Template](https://github.com/JLogical-Apps/flood-template)
2. Follow the setup instructions provided in the template's README.md file to initialize your project.
